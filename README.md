# ts-koa-passport-mongo-example

Auth microservice example.

TypeScript, Passport, Koa.

To get up and running:

```shell
git clone
cd auth
npm ci
npm start
# build; runs tests
npm run build
# test
npm t
# production server
node .
```

Includes examples for Google CloudBuild, CircleCI, Docker, and Kubernetes.
